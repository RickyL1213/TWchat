命令行下的微信客户端

![demo](./TWchat/demo.gif "demo")

目前在 macOS 上进行开发

理论上讲所有 Unix-like 均可用 (如有问题请提issue)

暂时只支持py2
py3暂不支持

## how to install 

```
pip install TWchat -U
```

install mpg123
 
### macOS

```
brew install mpg123
```

## how to use

```
twchat
```

命令行会出现二维码，扫码登录

``` 
ALT+q  exit twchat
```

语音消息需要安装mpg123

支持图片 语音消息 该类型消息默认会存储到 HOME 目录的 .twchat
